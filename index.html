Quick Start

Running in under 60 seconds

1
Clone the repository
git clone https://github.com/deep1ne8/misc.git && cd misc
2
Open PowerShell as Administrator and bypass execution policy for this session
Set-ExecutionPolicy -Scope Process Bypass
3
Launch AutoByte
.\AutoByte.ps1
4
Select a module by number and follow prompts. Logs write to:
C:\ProgramData\AutoByte\Logs\

Extensible

Add your own modules

Three steps to add a new module. The pattern is consistent — any PowerShell engineer can pick it up immediately.

AutoByte.ps1 — add a module
# 1. Write your function
function Invoke-YourModule {
    Invoke-WithLog 'Description of what this does' {
        # your code here
        Write-Log 'Done.' -Level OK
    }
}
 
# 2. Add label to $menuItems
$menuItems += 'Your Module Label'
 
# 3. Add switch case
'11' { Invoke-YourModule }
